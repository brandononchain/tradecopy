Alert Message Templates
Copy into TradingView alert message box
✓ For Pine Script strategies — symbol, direction, and size are all dynamic. Works for any instrument automatically.
"action": "{{strategy.order.action}}", "symbol": "{{ticker}}", "qty": "{{strategy.order.contracts}}", "price": "{{close}}", "sl": "{{plot_0}}", "tp": "{{plot_1}}", "comment": "{{strategy.order.comment}}"
{{strategy.order.action}} — auto buy/sell from strategy
{{ticker}} — chart symbol (US100, EURUSD, XAUUSD…)
{{strategy.order.contracts}} — lot size from strategy order
{{plot_0}} / {{plot_1}} — optional SL/TP from plotted values
ℹ For indicator alerts — create one alert for BUY and one for SELL. Symbol is still dynamic via {{ticker}}.
BUY Alert
"action": "buy", "symbol": "{{ticker}}", "qty": "1", "price": "{{close}}"
SELL Alert
"action": "sell", "symbol": "{{ticker}}", "qty": "1", "price": "{{close}}"
Change "qty":"1" to any lot size. {{ticker}} auto-fills from whichever chart fires the alert.
ℹ Sends a close action to exit an open position on that symbol.
"action": "close", "symbol": "{{ticker}}", "qty": "0"
